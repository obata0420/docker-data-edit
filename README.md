# 初期セットアップ
```
$ git clone https://github.com/obata0420/docker-data-edit.git
$ cd docker-data-edit
```
# 初期コンテナ構築
```
$ make init
```

# コンテナの構成
```
appコンテナ、webコンテナ、dbコンテナの３つのコンテナで構成されている。
それぞれの詳細は下記となる。

## appコンテナ
Laravelを動作させているコンテナ。
phpはバージョンは8.1.7
Laravelのバージョンは9.16.0

## webコンテナ
nginxとreactを動作させているコンテナ。
nginxのバージョンは1.22
reactのバージョンは17.0.2

## dbコンテナ
MySQLを動作させているコンテナ。
MySQLのバージョンは8.0
```

# 構築される環境について
```
## 概要
タスクの登録、編集、削除、既存タスクの検索が行うことができる。
各種ボタンの説明は下記。

### 検索
タスク名を入力し、押下したら入力文字の部分一致で検索が行われる。

### 検索リセット
検索内容をリセットし、初期表示に戻すボタン

### スケジュール登録
スケジュールの登録を行うボタン。

### 編集
該当の列のタスクの編集を行うことができるボタン。

### 完了
該当の列のタスクを完了とし、表から削除する。
```
